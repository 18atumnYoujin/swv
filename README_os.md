설치 시작을 누르면 사용자 설정으로 넘어갑니다.


 ![](images/SWV/centos7_user.JPG)

 OS가 설치되는 중에 root 암호와, 사용자를 생성하며 완료 후 재부팅을 합니다.  

 재부팅이 되면 라이센스 동의 목록을 체크한 후 os 설정을 완료합니다.
